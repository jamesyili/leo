Ability to [configure different API keys for models based on their purpose](https://github.com/datasette/datasette-llm/blob/0.1a4/README.md#model-references-with-custom-api-keys) - for example, set it up so enrichments always use `gpt-5.4-mini` with an API key dedicated to that purpose. [#4](https://github.com/datasette/datasette-llm/pull/4)

I released [llm-echo 0.3](https://github.com/simonw/llm-echo/releases/tag/0.3) to provide an API key testing utility I needed for the tests for this new feature.
