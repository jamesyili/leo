**Release:** [datasette-enrichments-llm 0.2a0](https://github.com/datasette/datasette-enrichments-llm/releases/tag/0.2a0)

    

This plugin now uses [datasette-llm](https://github.com/datasette/datasette-llm) to configure and manage models. This means it's possible to [specify which models](https://github.com/datasette/datasette-enrichments-llm/blob/0.2a0/README.md#configuration) should be made available for enrichments, using the new `enrichments` purpose.
